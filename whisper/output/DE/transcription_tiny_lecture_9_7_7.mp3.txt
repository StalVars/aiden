  Der Agent kann bestimmte Eigenschaften der Welt wahrnehmen. Wenn er einen Gestank auf einer Zelle bemerkt, dann weiß er, dass der Wumpus neben dieser Zelle ist. Ein Luftzug weist auf eine nahelgelegene Grube hin. Befindet sich der Agent auf der Zelle mit dem Gold, so bemerkt er ein Glitzern. Der Agent kann keine Wände sehen, aber er spürt die Stoß, wenn er versucht, gegen eine Wand zu laufen. Der Agent weiß, wann er den Wumpus erschossen hat, denn dann hört er einen Schrei.
